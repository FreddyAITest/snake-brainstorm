## 🤖 Automation Setup

**Cron Job:** Läuft jede Minute (`* * * * *`)  
**Browser:** Headless Chromium (OpenClaw)  
**Tracking:** Automatische Domain-Erkennung + Logging  
**Output:** Markdown-Datei + GitHub Push

### Config
```json
{
  "schedule": "* * * * *",
  "session": "isolated",
  "browser": "headless",
  "delivery": "none"
}
```
